ℹ️ Tentang LeksaLaw
Platform belajar hukum offline berbasis HTML statis

📖 Apa itu LeksaLaw?

LeksaLaw adalah sistem manajemen pembelajaran (LMS) offline yang dirancang khusus untuk mahasiswa Ilmu Hukum. Seluruh aplikasi berjalan sepenuhnya di browser tanpa membutuhkan koneksi internet, server, database, atau instalasi apa pun — cukup buka index.html.

✨ Fitur Utama

  • Kuis interaktif dengan Mode Belajar (feedback instan) & Mode Ujian
  • Pengacakan soal dan pilihan jawaban
  • Navigasi soal, tandai ragu, dan progress tracking
  • Penyimpanan nilai dan riwayat belajar otomatis (Local Storage)
  • Sistem favorit untuk quiz yang sering diakses
  • Pencarian cepat mata kuliah & quiz (Ctrl+K)
  • Mode gelap (Dark Mode)
  • Tampilan responsif — nyaman di desktop, tablet, dan ponsel

🛠️ Teknologi

Dibangun murni dengan HTML, CSS, dan JavaScript vanilla (tanpa framework, tanpa build tool). Data progress disimpan secara lokal menggunakan Web Storage API milik browser, sehingga privasi pengguna terjaga dan aplikasi tetap dapat dijalankan dari flashdisk maupun folder lokal mana pun.
